    2. 新节点=>种子节点 ： 我来了，我的Ip是xx 端口是yy

全体sayhi,只为加上白名单：
    1. 新节点给所有已知节点发消息，新节点的网件上把所有节点加上白名单
    2. 已知节点给新节点发消息，已知节点加白名单
